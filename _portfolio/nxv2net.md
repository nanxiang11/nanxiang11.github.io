NxV2Net 是 V2Net 裂缝分割工作的科研代码仓库，围绕复杂真实场景中的裂缝分割鲁棒性展开。

* 提出嵌套多尺度网络结构，增强裂缝细粒度几何特征表达。
* README 中展示了 V2Net 动机、结构贡献、SUES-CRACK 数据集和多数据集结果。
* 适合作为科研成果与工程复现入口。
